# TOTO - Your City, Your Ride

This is a NextJS starter for the TOTO ride-booking platform, built with Firebase Studio.

## Features
- **Rider App**: Book e-rickshaws and cabs with real-time OTP verification.
- **Driver Hub**: Manage earnings, track trips, and toggle online status.
- **Admin Console**: Real-time system monitoring, financial ledger, and user management.
- **Integrated Backend**: Uses Firestore for real-time data and Firebase Auth for secure login.

## Tech Stack
- **Frontend**: Next.js 15 (App Router), React 19, Tailwind CSS.
- **UI Components**: ShadCN UI, Lucide Icons.
- **Backend**: Firebase Firestore, Auth, Storage.
- **AI**: Genkit (Ready for integration).
