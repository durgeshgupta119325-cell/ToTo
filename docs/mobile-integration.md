
## 3. Initialization Code
In your `lib/main.dart`:

```dart
import 'package:flutter/material.dart';
import 'package:firebase_core/firebase_core.dart';
// import 'firebase_options.dart'; // Generated by FlutterFire CLI

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(
    // options: DefaultFirebaseOptions.currentPlatform,
  );
  runApp(const MyApp());
}
```

## 4. Security Rules Note
The Firestore and Storage rules are already set up to handle `customer` and `driver` roles. Make sure your Flutter app's Auth logic correctly sets these roles in the `users` collection upon sign-up.
